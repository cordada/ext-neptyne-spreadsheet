Neptyne allows you to build UIs in Google Sheets using Streamlit. Simply use the @streamlit decorator like this:

{% apply highlight_code %} {% whitespace all %} import neptyne as nt import streamlit as st @nt.streamlit(title="My Streamlit App", width=640, height=480) def my_streamlit_function(): st.write("Hello, world!") {% whitespace single %} {% end %}

Once you have this decorator in your code, the app will show up under this menu item. You can use the decorator with or without arguments. Possible arguments are:

  • title: The title of the Streamlit app.
  • width: The width of the Streamlit app.
  • height: The height of the Streamlit app.
  • sidebar: Whether to show the app in the sidebar.
  • auto_open: Whether to open the app automatically when the sheet is opened

You can find more information about Streamlit at streamlit.io.
